# UI Mockup Context for Google Stitch

# Product Vision

Design a modern web application for Technical Writers that feels like
GitHub, Linear, and Vercel.

Avoid playful education aesthetics. This is a professional daily
learning platform.

# Keywords

Minimal Professional Developer-first Clean Modern Fast Accessible
Monochrome Subtle animations Desktop-first with responsive mobile
support

# Color Direction

-   Neutral grayscale
-   GitHub-inspired light and dark themes
-   Minimal accent color (blue) only for primary actions
-   No gradients
-   No glassmorphism
-   No cartoon illustrations

# Typography

-   Clean sans-serif
-   Large hero heading
-   Comfortable reading width
-   Documentation-inspired spacing

# Dashboard

Hero: "Sharpen your documentation skills. Every day."

Supporting copy: Practice interview-ready questions, master
documentation skills, and stay current with AI and Docs-as-Code.

Primary CTA: Start Practicing

Sections: - Continue Practice - Daily Challenge - Quick Practice
(5/10/20) - Practice by Skill - Random Quiz - Recent Performance

# Practice by Skill

Display skills as clean bordered cards.

Each card contains: - Skill name - Short description - Question count -
Start button

# Quiz Setup

Controls: - Question count (5/10/20) - Difficulty
(Easy/Medium/Hard/Mixed) - Timer (15/30/45/60)

Primary CTA: Start Quiz

# Quiz Screen

Header: Progress indicator Question counter Countdown timer Theme toggle

Main: Single question Four options Submit button

After submission: Correct/Incorrect status Explanation Reference
Real-world example

Automatically continue after a short delay.

# Results

Display: Score Accuracy Average response time Incorrect questions Retry
Back to Dashboard

# Design Principles

-   Everything should fit an engineer's workflow.
-   White space over decoration.
-   Cards with subtle borders.
-   Rounded corners (6-8px).
-   Soft shadows only where necessary.
-   Focus on readability.
-   Optimized for static website implementation.

Generate desktop and mobile mockups for all MVP screens with consistent
components and spacing.
