# Product Requirements Document (PRD)

## Project Name

**Working Title:** Technical Writing Practice Platform

**Tagline:** Sharpen your documentation skills. Every day.

------------------------------------------------------------------------

# Vision

Build the best daily learning platform for Technical Writers by
combining interview preparation, skill development, and modern
documentation practices into a simple, fast, GitHub-inspired web
experience.

------------------------------------------------------------------------

# Problem Statement

There is no single platform dedicated to helping Technical Writers
consistently improve their skills through structured daily practice.

Existing quiz platforms are generic, outdated, or focus only on
certifications rather than real-world documentation work.

This platform aims to fill that gap.

------------------------------------------------------------------------

# Goals

## Primary Goals

-   Help users improve documentation skills through daily practice.
-   Prepare users for Technical Writing interviews.
-   Encourage continuous learning.
-   Build habits through short quizzes.
-   Keep pace with evolving technologies like AI, Docs-as-Code, and API
    documentation.

## Non-goals (MVP)

-   User accounts
-   Global leaderboard
-   Cloud sync
-   Paid subscriptions
-   AI-generated answers at runtime

------------------------------------------------------------------------

# Target Audience

-   Aspiring Technical Writers
-   Entry-level Technical Writers
-   Experienced Technical Writers
-   API Documentation Specialists
-   Information Developers
-   UX Writers
-   Content Designers
-   Documentation Engineers
-   Professionals preparing for interviews

------------------------------------------------------------------------

# MVP Scope

## Included

-   Daily dashboard
-   Practice by skill
-   5/10/20 question quizzes
-   30-second default timer (user configurable)
-   Immediate feedback after every question
-   Explanations
-   References
-   Real-world examples
-   Static JSON question bank
-   Responsive UI
-   GitHub Pages deployment
-   Google Analytics 4
-   Microsoft Clarity

## Excluded

-   Login
-   Leaderboard
-   Streak sync
-   User profiles
-   Backend database

------------------------------------------------------------------------

# Success Metrics

-   Quiz completion rate
-   Average score
-   Average time per question
-   Return visitors
-   Daily challenge participation
-   Most practiced skills
-   Most difficult questions

------------------------------------------------------------------------

# Technical Constraints

-   Static site
-   GitHub Pages hosting
-   Vanilla React or Next.js static export
-   No paid infrastructure
-   Browser local storage for preferences
